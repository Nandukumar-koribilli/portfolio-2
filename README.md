# Nandu Kumar Koribilli Portfolio

A futuristic personal portfolio website featuring glassmorphism design and high-quality 3D animations.

## Project Structure

This repository contains two implementations of the portfolio:

### Root (Vanilla JS)
The root directory contains a Vanilla JavaScript implementation built with:
- **GSAP** for animations
- **HTML5 Canvas** for particle effects and sequential scroll animations

### `portfolio-react/` (React)
A React-based version with a modern tech stack:
- **React 19** with **TypeScript**
- **Framer Motion** for animations
- **Tailwind CSS v4** for styling
- **OGL** for advanced WebGL/3D animations

## Tech Stack

| Category      | Technologies                                      |
|---------------|---------------------------------------------------|
| Frontend      | React 19, TypeScript, Vanilla JS                  |
| Styling       | Tailwind CSS, CSS Custom Properties (Glassmorphism)|
| Animations    | GSAP, Framer Motion, OGL                          |
| Build Tool    | Vite                                              |

## Getting Started

### Prerequisites
- [Node.js](https://nodejs.org/) (v18 or later recommended)
- npm

### Clone the Repository
```bash
git clone https://github.com/Nandukumar-koribilli/portfolio-2.git
cd portfolio-2
```

### Run the Vanilla JS Version (Root)
```bash
npm install
npm run dev
```

### Run the React Version
```bash
cd portfolio-react
npm install
npm run dev
```
